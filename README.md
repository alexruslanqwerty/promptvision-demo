# PromptVision (ДЕМО-ВЕРСИЯ)

Добро пожаловать в демонстрационную версию программы **PromptVision**!

Эта папка содержит урезанную, ознакомительную версию программы, которая работает прямо в вашем браузере. Вам **не нужно** ничего скачивать, устанавливать или настраивать.

## 🚀 Как запустить ДЕМО?
Просто дважды кликните по файлу **`index.html`** в этой папке. Он откроется в вашем браузере (Chrome, Safari, Edge и т.д.).

## 💡 Что умеет демо-версия?
- **Интерфейс:** Вы можете полностью оценить внешний вид программы, понажимать все кнопки и посмотреть, как работает выбор стилей и параметров (Refinements).
- **Загрузка фото:** Вы можете загружать любые фотографии (перетаскиванием или через `Ctrl+V`).
- **Имитация нейросети:** При нажатии кнопки "Анализировать", программа реалистично сымитирует процесс раздумий и выдаст один из заготовленных высококачественных примеров промптов. 
- **Галерея:** Вы можете нажимать кнопку "Сохранить", и результаты будут сохраняться в Галерею прямо в памяти вашего браузера.

## ⚠️ Отличия от полной версии
В этой ДЕМО-версии **отключен реальный искусственный интеллект**. 
В **полной версии** программы:
1. Нейросеть действительно распознает именно *вашу* фотографию, описывая каждую деталь (цвет глаз, одежду, фон).
2. Полная версия работает абсолютно приватно на вашей видеокарте через программу LM Studio (без интернета).
3. Доступна интеграция с настоящим Google Gemini API для молниеносной генерации текстов.
4. Доступен режим захвата кадров из видеофайлов.
